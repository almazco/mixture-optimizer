# Оптимизатор многокомпонентных смесей

Прототип системы оптимизации рецептур фармацевтических и химических смесей методом линейного программирования.

**Тема НИР:** Методы оптимизации многокомпонентных смесей  
**Автор:** Зиязетдинов Алмаз Радикович, НПИбд-01-22

### Возможности
- Минимизация стоимости смеси при заданных технологических и фармакологических ограничениях
- Автоматический расчёт оптимального состава
- Построение графика чувствительности стоимости
- Сохранение результатов в CSV и PNG

### Быстрый запуск

```bash
git clone https://github.com/твой_ник/mixture-optimizer.git
cd mixture-optimizer
pip install -r requirements.txt
python optimize_mixture.py
